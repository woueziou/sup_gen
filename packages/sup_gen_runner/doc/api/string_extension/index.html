extensions/string_extension library

String utility functions for case conversion and validation.

This library provides helper functions for converting between different naming conventions commonly used in database-to-code generation:

  • snake_case (database naming)
  • PascalCase (Dart class names)
  • camelCase (Dart property names)

Functions

isValidDartClassName(String text) → bool
Validates whether a string can be converted to a valid Dart class name.
snakeToCamelCase(String text) → String
Converts a snake_case string to camelCase.
snakeToPascalCase(String text) → String
Converts a snake_case string to PascalCase.
